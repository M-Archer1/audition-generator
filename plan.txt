# Scale Generator
Generate message for an audition that requests musician play certain scale in major or minor and a certain sight reading excerpt.

Player # 1 (etc, each time it is refreshed the player number should increase by 1)

## Random Elements
* instrument
* scale letter
* major or minor

### Sight reading
Options for Excerpts 1-5

### Scale Letters
- C, F, B-flat, E-Flat, A-flat, D-flat, G-flat
- G, D, A, E, B

Modules
- Data Module data.js
- Random number Module random.js
- Object Module object.js
- Main module for creating object with data and generate page content script.js
